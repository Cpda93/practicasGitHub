Sobre Comics-App

Comic-App es un prototipo que muestra las formas en las que podemos utilizar las nuevas tecnologías de desarrollo front-end.

  • Crear una tienda de productos, de giro general ya sea Ropa, mascotas, tecnología etc.
  • Utilizarla para buscar personas en una base de datos ya sea de una escuela, empresa o consumir una API.
  • Con Comic-App fomentamos la utilización de software libre como Linux y VisualStudio que es el entorno de desarrollo de esta Aplicación.
    • Comic-Aapp esta desarrollada con los siguintes lenguajes:
    1. Html 5
    2. Css 3
    3. JavaScript
    4. TypeScript
    5. Angular 7 (Framework)
    6. Boostrap 4 (Framework)
